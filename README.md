# angularjs-ts-boilerplate

A boilerplate using typescript to develop angularjs. 

- TsLint
- Translate (i18n)
- Reset CSS
- PostCSS

## Dependencies

- angularjs ^1.7.8
- @uirouter ^1.0.22
- webpack ^4.29.6
- typescript ^3.3.4000

## development

	git clone --depth=1 https://github.com/mateusmirandaalmeida/angularjs-ts-boilerplate.git your-project-name

	cd your-project-name

	npm install

	npm run dev

## production

    npm run prod

    Your project -> /public
